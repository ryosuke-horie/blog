## 課題

リモートリポジトリ（GitHub）で削除したブランチがローカル環境に残っている問題について。VSCode上に消したはずのブランチが表示されると、整理したくなります。

## 技術的背景

Git ではリモートブランチとローカルブランチの管理は別々になっているため、明示的にコマンドを実行する必要があります。

## 解決方法

実行するコマンド：

```bash
git fetch origin --prune
```

## オプションの説明

`--prune` フラグは、リモートリポジトリで削除されたブランチの情報をローカルリポジトリから削除する機能を提供します。

この単純なコマンド実行により、リモートでの削除がローカル環境に正しく反映されます。
